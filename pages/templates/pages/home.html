About Site:

PasaleDai™ is an online shopping platform dedicated to supplying everyday products and fresh groceries at every customer's doorstep.



Docs:

User classes:

  • User(Normal): Can write, edit or delete blog posts. Cannot create, edit or delete product listings.
  • Seller: Can create, edit or delete product listings. Requires invite code to signup.
  • Admin: Can do everything. Can generate custom InviteCode for seller signups.

Apps:

  • accounts: Create and manage user accounts
  • cart: Create and manage customer's cart
  • blogs: Create and manage user blogs
  • pages: Home, About, Contact-us
  • products: Create and manage seller products
